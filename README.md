# Delailah Resto API

## Descripcion:
API de gestion de pedidos de un restaurante, realizado para el primer sprint proyect de Acamica. 

## Teconologias utilizadas:
- NodeJS
- fs
- Express
- Swagger
- Dotenv

## Instalacion:

```bash
git clone
npm install
npm run dev
```

## Uso

http://localhost:3000/api-docs/
